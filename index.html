Intro

LiteroticAPI is an API to Literotica, in order to download stories and keep them on your own computer without having to depend on Literotica. I'm not entirely sure if anyone will find any use for it, but who knows.

Install

LiteroticAPI will be found on PyPI soon, so just run a quick

pip install LiteroticAPI

to grab it.
